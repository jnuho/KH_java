배열

자바에서의 배열과 비슷함. 거의 arrayList와 동일 배열 생성 : [], new Array()를 이용하는 방법이 있음 배열의 길이는 자동으로 증가함. 접근하거나 값을 대입할때는 인덱스를 이용

배열 메소드활용

indexOf : 배열의 요소가 위치하고 있는 index번호를 리턴

concat : 두개이상의 배열을 합치는 기능

join : 배열을 합치는 작업 -> 문자열로 합치는 것

문자열로 합칠때 사용할 구분자를 매개변수로 넣어줌

sort : 배열을 내림차순, 오름차순으로 정렬해주는 기능

배열의 값을 수정하는 함수

push()/pop() push : 배열의 맨뒤에 값을 추가하는 기능 pop : 배열의 맨뒤에 값을 삭제하는 기능

unshift/shift

unshift : 배열의 맨앞에 값을 추가
shift:배열의 맨앞의 값을 삭제

slice/splice

slice(number(index1),number(index2)) : 배열의 값(요소)선택해서 새로운배열 출력(substring와 비슷)
splice(start,deletecount,value) : 배열의 index위치의 요소를 제거하고 추가
