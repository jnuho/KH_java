DataStream을 이용하여 파일 입출력하기!!

새로운 프로젝트를 생성하여 하세요

data.com.controller패키지에 DataStream클래스를 생성

DataStream에 데이터를 파일에 입력할 addFile()를 생성하여 데이터를 
제품명, 가격, 수량, 할인율(본인이 자유롭게 정하세요)을 product.dat라는 파일에 
넣어보자

파일의 데이터를 출력할 printFile()을 생성하여 
product.dat의 내용을 아래와 같이 출력하자

===== 저장 물품 =======
제품명 : 0000
가격 : 0000원
수량 : 00개
할인율 : 0.00%

~~~  저장 물품을 5개를 입력받고 파일에 저장 후  출력하세요!

data.com.vo에 Product클래스를 만들고 이것을 이용하여 입출력해보세요~
